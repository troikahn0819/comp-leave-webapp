
보상휴가 자동계산 웹앱 — PRD (초안 v0.1)

본 문서는 인터뷰 진행 중 작성된 초안입니다. 일부 항목은 [미정] 으로 표시되어 있으며, 이후 확정 후 갱신합니다.


1. 개요 / 목적
근태 담당자가 각종 초과근무에 대한 보상휴가를 산정해 근태시스템에 입력하는 작업을 현재 수기로 처리하고 있다. 본 웹앱은 이 산정 과정을 자동화하여 검토·계산 부담을 줄이는 것을 목표로 한다.
현재 업무 흐름

근태시스템에서 초과근무 결과보고 내용을 종류별로 엑셀(.xls) 다운로드
담당자가 엑셀을 검토하고 보상휴가 시간을 수기로 계산
계산 결과를 근태시스템에 다시 입력

본 웹앱이 담당하는 범위: 2번(검토 + 보상휴가 계산). 즉 엑셀 입력 → 보상휴가 산정 → 재입력용 결과 출력.

2. 입력 데이터

엑셀 1개 = 초과근무 종류 1개 (종류별로 따로 다운로드하므로 파일 자체가 종류를 구분함)
파일 형식: .xls
공통 열(헤더): 사번, 이름, 부서, 일자, 시작시간, 종료시간
데이터 형식

일자: 2000-01-01 (YYYY-MM-DD)
시작시간 / 종료시간: 2000 형태(HHMM 4자리)
자정 넘김: 종료시간에 당일/익일 정보가 포함됨. (없을 경우 종료 < 시작이면 익일로 보완 처리)




3. 초과근무 종류별 계산 규칙 (핵심 로직)
총 5종류. 종류별 계산식이 다르다.
① 일반연장근무

가산율: 1.5배
휴게 공제: 근무 4시간 이상이면 30분 공제
절사: 10분 단위 절사
시간 제한: 신청은 22시까지만 가능. 종료가 22시를 넘기는 경우 22시로 절단하며, 그 이후 근무는 철야근무로 별도 신청
야간(22~06): 인정하지 않음

② 휴일근무 (교대근무자만 해당)

가산율: 1.5배
휴게 공제: 없음
근무 형태: 교대근무 특성상 전일(8시간) / 반일(4시간) 두 가지만 존재 (애매한 중간값 없음)
전일/반일 판정: 시작·종료시간으로 산출한 근무시간 기준 (8h → 전일, 4h → 반일)
적치 결과: 전일 = 8 × 1.5 = 12시간, 반일 = 4 × 1.5 = 6시간
야간 가산: 불필요 (야간수당은 별도 계산)
구분 의미: 해당일이 공휴일인 경우

③ 대체근무

계산식은 휴일근무와 완전히 동일 (1.5배 / 휴게 없음 / 전일 12h·반일 6h)
차이점: 해당일이 공휴일이 아님(평일에 수행하는 대체근무)
종류별 파일이 분리되어 있으므로 계산 자체에는 영향 없음

④ 비상출동

발생 시점에 제한 없음 (언제든 발생, 자정 넘김 가능)
시간대 분할 가산: 주간(06~22) 1.5배, 야간(22~06) 2배. 한 건이 두 시간대에 걸치면 구간별로 나눠 곱한 뒤 합산
휴게 공제: 4시간마다 30분. 공제 위치 = 근무 시작 후 4시간 경과 지점이 속한 시간대에서 차감 (8시간 근무면 4시간·8시간 지점에서 각각 30분씩 반복 공제)
절사: 신경 쓰지 않아도 됨 (결과보고 입력이 10분 단위로만 가능하여 자연 정리됨)
예시: 20:00~익일 02:00(총 6h) → 주간 2h(20~22시), 야간 4h(22~02시), 휴게 30분은 4시간 경과 지점(24:00, 야간)에서 차감 → 야간 3.5h → 2×1.5 + 3.5×2 = 10시간

⑤ 철야근무 (일근직만 해당, 평일 근무)

시간대: 무조건 야간(22~06), 기본 가산율 2배
휴게 공제: 4시간마다 30분 → 먼저 공제하여 "철야시간" 산출
익일 자동휴가 차감 로직

익일이 평일이면: 철야시간만큼 다음날 자동휴가 사용 → 순적치 = 철야시간 × 1
익일이 평일이 아니면(휴일·휴무): 자동휴가 사용 불가 → 차감 없이 철야시간 × 2 적치


시스템 입력값: 위에서 계산한 순적치만 한 줄로 입력

예) 익일 평일 + 철야 4h → 4 입력 / 익일 휴일 + 철야 4h → 8 입력


익일 기준: 일자 + 1일
휴일여부 판단: 별도 근무표(날짜 기준, 전 직원 공통) 에서 조회. 값은 평일 / 휴일 / 휴무 3종 → "평일이 아니면 ×2" 로 단순 적용


4. 근무표(휴일여부) 데이터

철야근무의 익일 휴일여부 판단에 사용
조회 키: 날짜만 (철야는 일근직만 해당하여 전 직원 공통)
값: 평일 / 휴일 / 휴무
운영 방식(안): 보유 중인 연간 근무표 엑셀을 한 번 업로드하면 날짜 → 상태로 저장해두고 철야 계산에 자동 활용. 갱신 시 재업로드.
[미정] 근무표 엑셀의 구체적 열 구성(열 이름 등)


5. 출력 / 재입력 [미정 — 인터뷰 중단 지점]

표시 항목(예: 사번 / 이름 / 부서 / 보상휴가 적치시간) — 확정 필요
합산 단위(직원별 합산 vs 건별 표시) — 확정 필요
내보내기 방식(화면 복사 vs 결과 엑셀 다운로드) — 확정 필요
근태시스템 재입력 형식(정해진 양식 여부) — 확정 필요


6. 그 밖에 확정이 필요한 사항 (Open Questions)

실행 환경: claude.ai 아티팩트로 사용 vs HTML 파일로 받아 회사 PC에서 단독 실행 → 개인정보 처리/저장 방식 결정에 영향
데이터 저장: 근무표 등 영속 저장 필요 여부 및 방식
시간 셀 처리: 시작/종료시간 셀이 텍스트인지 숫자인지(앞자리 0 처리), 0200 등 표기 확인
당일/익일 정보 위치: 별도 열인지, 종료시간 셀 내 표기인지
휴게 "4시간마다 30분"의 장시간 근무(12h+) 반복 공제 확인
