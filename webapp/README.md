# 보상휴가 자동계산 웹앱

Stitch 디자인 6종을 **디자인 변경 없이** 동작하는 웹앱으로 구현한 결과물입니다.
초과근무 엑셀을 업로드하면 PRD v0.1의 5종 계산 규칙에 따라 보상휴가 적치시간을
자동 산정하고, 재입력용 결과를 엑셀/CSV로 내려받을 수 있습니다.

## 실행 방법

별도 빌드/설치가 필요 없는 정적 HTML 앱입니다. 두 가지 방법 중 하나로 실행합니다.

1. **간단 실행**: `webapp/index.html` 더블 클릭 (브라우저에서 바로 열림)
2. **로컬 서버**(권장, 일부 브라우저의 파일 보안정책 회피):
   ```
   npx http-server webapp -p 5173
   ```
   접속: http://localhost:5173

> 인터넷이 필요합니다 — TailwindCSS·폰트·SheetJS(xlsx)를 CDN에서 로드합니다.
> 사내 폐쇄망에서 쓰려면 해당 리소스를 로컬에 내려받아 경로만 바꾸면 됩니다.

## 페이지 구성

| 파일 | 화면 | 기능 |
|------|------|------|
| `index.html` | 대시보드 | 요약 KPI, 빠른 시작 버튼 → 각 페이지 이동 |
| `upload.html` | 엑셀 업로드 | **핵심.** .xls/.xlsx 업로드 → 자동 계산 → 결과 표시·다운로드 |
| `results.html` | 계산 결과 | 저장된 결과 조회·필터·엑셀/CSV 다운로드 |
| `schedule.html` | 근무표 관리 | 연간 근무표 업로드(날짜→평일/휴일/휴무), 달력 미리보기 |
| `rules.html` | 계산 규칙 | 5종 규칙 카드 + 엔진 기반 예시 계산 |
| `history.html` | 다운로드 내역 | 생성한 결과 파일 이력 |

## 계산 엔진 (`engine.js`)

PRD 3장의 5종 규칙을 구현합니다.

- **① 일반연장근무**: 1.5배 / 4h↑ 30분 공제 / 10분 절사 / 22시 절단(야간 불인정)
- **② 휴일근무**: 교대근무, 전일 8h→12h·반일 4h→6h
- **③ 대체근무**: 휴일근무와 동일(평일 수행)
- **④ 비상출동**: 주간(06~22) 1.5배·야간(22~06) 2배 구간 분할, 4h마다 30분 공제
- **⑤ 철야근무**: 야간 2배, 4h마다 30분 공제→철야시간, 익일 평일이면 ×1·휴일/휴무면 ×2
  (근무표 조회, 미업로드 시 주말=휴일로 추정)

### 입력 엑셀 형식
공통 헤더: `사번, 이름, 부서, 일자, 시작시간, 종료시간`
- 일자: `YYYY-MM-DD`
- 시작/종료시간: `HHMM`(예 `1800`), 자정 넘김은 `익일0200` 또는 종료<시작 시 자동 보완
- 파일명에 근무종류가 포함되면 자동 인식(예 `일반연장근무_2026-06.xls`)

## 데이터 저장
업로드한 근무표·계산 결과·다운로드 이력은 브라우저 `localStorage`에 저장됩니다
(서버 전송 없음 → 개인정보가 PC 밖으로 나가지 않음).
